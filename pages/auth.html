Description

A web authentication form for your apps

Required softwares

  • XAMPP

Used Technologies

  • DB: MARIADB

  • Web application: PHP, HTML

How to install

To install this project you've to:

  • Run the following command inside your terminal: git clone https://github.com/teachermarchesan/ps-cafa-leonardo.git

  • Move the cloned repository to your htdocs folder

  • Start XAMPP

  • Then Navigate to: http://localhost/TPSIT/ps-cafa-leonardo/AUTH

XAMPP will automaticaly find the index.php file inside the folder and will run the web application.

Usage

The user is provided two choices:

  • Sign up: the user can register a new account into the web application

  • Login: the user can log in with an registered account

Once logged in the user is granted access to his private dashboard, in which is also provided a button to log out.
