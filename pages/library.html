Description

The web app to handle every aspect of your library

Required softwares

  • XAMPP

Used Technologies

  • DB: MARIADB

  • Web application: PHP, HTML

How to install

To install this project you've to:

  • Run the following command inside your terminal: git clone https://github.com/teachermarchesan/ps-cafa-leonardo.git

  • Move the cloned repository to your htdocs folder

  • Start XAMPP

  • Then Navigate to: http://localhost/TPSIT/ps-cafa-leonardo/LIBRARY

XAMPP will automaticaly find the index.php file inside the folder and will run the web application.

Usage

The user is provided a books' table and a form to search a book by its title.

The user can also log in to request the loaning of books. The administrator will then authorizes the loan from his dashboard.
